# Máy tính Chu kỳ Ngủ – PRO v6.1

Công cụ tính toán giấc ngủ lý tưởng dựa vào chu kỳ ngủ. Hỗ trợ:
- Tính giờ đi ngủ / thức dậy
- Ghi nhận giấc ngủ thực tế
- Đánh giá theo chu kỳ

Tối ưu cho iPhone: có thể thêm vào màn hình chính như ứng dụng PWA.

## Cách sử dụng
1. Tải toàn bộ file lên GitHub hoặc GitLab repository.
2. Bật GitHub Pages hoặc GitLab Pages.
3. Truy cập trang web qua đường dẫn được cung cấp.
